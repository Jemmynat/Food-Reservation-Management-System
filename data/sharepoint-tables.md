# SharePoint Data Tables

The Food Reservation Management System uses four SharePoint lists to store and manage application data.

---

## 1️⃣ Reservations
Stores all food reservation records.
- Reservation ID
- Customer Name
- Reservation Date
- Reservation Status
- Assigned Employee

---

## 2️⃣ Employees
Stores employee information.
- Employee Name
- Role
- Contact Details
- Availability Status

---

## 3️⃣ Admin Data
Stores administrative configuration and controls.
- Admin Name
- Permissions
- System Settings

---

## 4️⃣ Reference / Lookup Data
Supports dropdowns and validation within the app.
- Status values
- Categories
